Flex lexer folder contains g++ lexer C Program												 *
To Execute:																				 	 *
	Go to the this folder and run make file 												 *
	And then you can run with 2 options:													 *
																							 *
		"./gpp_lexer" (REPL mode) output in the parsed_cpp.txt file							 *
		"./gpp_lexer" absolute-file-path" (Read file mode) output in the parsed_cpp.txt file *
																							 *
**********************************************************************************************
																							 *
Lisp lexer folder contains g++ lexer Common Lisp Program									 *	
To Execute:																					 *
	Go to the this folde and run gpp_lexer.l file											 *	
	You can run with 2 options:																 *
																							 *
		"clisp gpp_lexer.l" (REPL mode) output in the parsed_lisp.txt file					 *
																							 *
**********************************************************************************************
																							 *
To exit flex or lisp program you should give input 'enter' twice							 *
For example:																				 *
	You give input (+ 3 2) and you give enter twice and program finished.					 *
	And then program write result to output file											 *
